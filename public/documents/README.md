# Documents Folder

Place your CV PDF file here with the name `cv.pdf`

## Instructions

1. Save your resume/CV as a PDF file
2. Rename it to exactly: `cv.pdf`
3. Place it in this folder (`public/documents/`)
4. The CV section on your website will automatically link to this file

The download button will automatically use the filename "Camilo_Otalora_CV.pdf" when users download it.
